1. **Background Configuration**:
   - Use 'clearColor' for most space and abstract environments

   - Enable fog to enhance depth and atmosphere
   - Fog color should complement the background (RGB values 0-1)
   - Start distance: 50-200 (closer for intimate settings, farther for open spaces)
   - End distance: 600-1200 (shorter for mysterious/claustrophobic, longer for open)
   - Mode: typically 'linear' for consistent effects

4. **Fog Color Guidelines**:
   - Choose fog colors that complement the world atmosphere
   - Consider the visual style and narrative mood
   - Use RGB values 0-1 that enhance depth without overwhelming gameplay

5. **Lighting Configuration**:

   - Direction: Typically [0, 1, 0] to [0.3, 1, 0.5] for natural top-down lighting
   - Position: [0, 40-80, 0-30] for overhead positioning

   - Specular color: Usually similar to diffuse but can be different for special effects

6. **Lighting Color Guidelines**:

   - Balance illumination with atmospheric immersion
